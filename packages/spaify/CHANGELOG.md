# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/2.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0]

- Add progressive enhancement support for the View Transition API.
- Add `keepHtmlAttributes` and `keepBodyAttributes` options for syncing page attributes during navigation.
- Improve script cloning to preserve script attributes.
- Improve navigation handling for modified clicks, downloads, non-self targets, same-page links, and failed fetches.

## [0.0.8]

- Update build scripts.

## [0.0.3]

- Support nested anchor tags.

## [0.0.2]

- Add script handling.

## [0.0.1]

- Initial release.
